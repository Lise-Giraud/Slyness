Partage de niveaux

Cette page est dédiée au partages des niveaux que vous créez en créatif. Vous pouvez ici partagez les fichiers en .corn correspondants au niveau que vous créez afin que d'autre joueurs puissent les récupérer et y jouer à leur tour ! De-même, il est possible de télécharger des niveaux afin d'y jouer dans le mode créatif !

Partagez vos niveaux

Partagez vos niveaux sur ce site : Slack.com
Pour récupérer des niveaux, cliquez sur l'onglet "fichier" sur la gauche de l'écran. Téléchargez ensuite les fichiers qui vous intéressent et voilà !
Pour partager vos niveaux, allez sur le canal de conversation "partage-de-niveaux" et envoyez les niveaux que vous voulez partager.

Comment récupérer les niveaux téléchargés ?

Une fois le jeu lancé, allez dans le mode créatif. Vous verrez une case avec écrit "déposez votre sauvegarde ici"

Il vous suffit juste de cliquer sur cette case et une fenêtre de vos téléchargements s'ouvrira. A partir de là, prenez le niveau que vous voulez importer et glissez-le dans la case et le niveau s'ajoutera tout seul à votre liste de niveaux dans le mode créatif ! Mais soyez sûr que votre fichier contenant le nivau soit bien un fichier en .corn ou sinon vous ne pourrez pas importer votre niveau dans le jeu.

Comment exporter un niveau ?

Pour exporter un niveau, nous n'avons pas encore un sytème très intuitif et très pratique. Comencez par vous rendre sur l'équipe Slack du jeu (le lien est donné plus haut sur la page). Une fois sur l'équipe, allez sur le canal appelé "partage-de-niveaux".

Une fois sur le canal, vous pouvez poster des messages et même joindre des fichiers. Cliquez donc sur le trombone qui sert à joindre un fichier, puis cliquez sur "Ajouter un fichier depuis votre ordinateur".

Un explorateur de fichier s'ouvrira alors. Une fois dessus, allez sur votre disque dur (très souvent il s'appellera Windows C:)

Une fois dans Windows C:, allez dans l'onglet "affichage" et cochez "Eléments masqués" et "extensions de noms de fichiers"

Ensuite, allez dans votre dossier Utilisateurs, puis le dossier contenant tous vos dossiers personnels (très souvent il sera nommé d'après votre prénom suivi de la première lettre de votre nom, ou quelque chose qui y ressemble).

Une fois dans ce dossier, vous verrez apparaître appelé AppData.

Allez dedans, puis cliquez sur le dossier LocalLow, puis Epicorne et enfin Slyness.
